# Restaurant Billing System

## Project Overview
A console-based Restaurant Billing System built with **Core Java**. 
This application allows restaurant owners to manage menu items and customers to place orders with automated billing including GST and discounts.

## Features

### Admin Functions
- Add new menu items
- Remove existing menu items  
- Update item prices
- View complete menu

### Customer Functions
- Browse restaurant menu
- Add items to order with quantities
- Apply discount (percentage-based)
- View itemized bill with GST calculation
- Save receipt to text file

## Technologies Used
- **Java** (Core Java)
- **Collections Framework** (ArrayList)
- **File I/O** (Receipt saving)
- **Java DateTime API**

## How to Run

### Prerequisites
- Java JDK 8 or higher installed

### Steps

```bash

git clone https://github.com/Praveen-J2003/Syntexhub_RestaurantBillingSystem.git

# Navigate to project folder
cd Syntexhub_RestaurantBillingSystem

# Compile all Java files
